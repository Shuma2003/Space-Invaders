# Space Invaders - Упрощенная версия
https://img.icons8.com/color/96/000000/space-invaders.png

Упрощенная браузерная версия классической аркадной игры Space Invaders, реализованная на чистом JavaScript, HTML и CSS.

## 🎮 Описание игры
Space Invaders - это классическая аркадная игра, где игрок управляет лазерной пушкой и должен уничтожить волны надвигающихся инопланетных захватчиков, прежде чем они достигнут Земли.

## Особенности этой версии:
🔫 Управление пушкой с помощью клавиатуры

👾 5 рядов врагов с интеллектуальным движением

💥 Система столкновений и уничтожения врагов

🏆 Система подсчета очков

🎯 Плавная анимация и отзывчивое управление

🔄 Возможность перезапуска игры

## 🚀 Как запустить игру
Скачайте файлы игры:

bash
git clone https://github.com/your-username/space-invaders.git
cd space-invaders
Откройте в браузере:

Просто откройте файл index.html в вашем браузере

Или используйте Live Server в VS Code

Играйте!

Нажмите пробел для начала игры

Управляйте пушкой с помощью стрелок ← →

## 🎯 Управление
Клавиша	Действие
←	Движение влево
→	Движение вправо
Пробел	Стрельба / Перезапуск игры
🎮 Игровой процесс
Цель игры:
Уничтожить всех врагов, стреляя в них из пушки

Не позволить врагам достигнуть нижней границы экрана

Набрать как можно больше очков

Механика:
Враги: Двигаются горизонтально как единая группа, опускаясь вниз при достижении края

Снаряды: Летят вертикально вверх, уничтожают врагов при попадании

Очки: +10 очков за каждого уничтоженного врага

Сложность: Скорость врагов увеличивается с каждым новым раундом

🛠 Технические детали
Стек технологий:
HTML5 - структура игры

CSS3 - базовые стили

JavaScript (ES6+) - игровая логика и рендеринг

Архитектура:
Модульная структура кода

Использование Canvas API для рендеринга

Игровой цикл на основе requestAnimationFrame

Система коллизий AABB (Axis-Aligned Bounding Box)

Основные функции:
gameLoop() - главный игровой цикл

updateEnemies() - логика движения врагов

checkCollisions() - система столкновений

drawScore() - отображение счета

🎨 Кастомизация
Вы можете легко изменить параметры игры, отредактировав константы в файле script.js:

javascript
// Настройки врагов
const enemyConfig = {
    rows: 5,
    cols: 10,
    speed: 1,
    // ... другие параметры
};

// Настройки игрока
const player = {
    speed: 5,
    // ... другие параметры
};
